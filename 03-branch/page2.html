Como fazer uma branch?


Exemplo de um terminal do Git Bash em operação.



Exemplo de um terminal do Git Hub aberto.

Via linha de comando:
1. Vá ao seu terminal (PowerShell/ Git Bash);
2. Digite dentro da pasta do projeto: 'git branch nome-da-sua-branch';
3. Acesse a branch: 'git checkout nome-da-sua-branch'.

Pelo github:
1. Abra seu repositório depois de fazer o fork/criar um novo repositório;
2. Clique em 'main' (antigo master) e digite o nome da branch que deseja criar.


E pronto, sua branch foi criada!
